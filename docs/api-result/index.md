# API Result

## 添加依赖

**Maven**

```xml
<dependency>
    <groupId>com.fengwenyi</groupId>
    <artifactId>api-result</artifactId>
    <version>3.0.0</version>
</dependency>
```

**Gradle**

```groovy
implementation 'com.fengwenyi:api-result:3.0.0'
```

或者

```groovy
implementation 'com.fengwenyi:api-result:+'
```
