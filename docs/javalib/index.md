# 快速开始

java 17

## 添加依赖

**Maven**

```xml
<dependency>
    <groupId>com.fengwenyi</groupId>
    <artifactId>JavaLib</artifactId>
    <version>3.0.0</version>
</dependency>
```

**Gradle**

```groovy
implementation 'com.fengwenyi:JavaLib:3.0.0'
```

或

```groovy
implementation 'com.fengwenyi:JavaLib:+'
```
